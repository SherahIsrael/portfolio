Challenges



One of the most challenging parts of this build was creating more than one modal on the same page. I used JQuery to fix this issue and called to the different modals directly from the HTML file, instead of calling to the ID's from the JS file.



Another challenge was getting the background image of the new products to be animated and responsive. I fixed this by using flexbox to separate the image and the div and adjusted the flex basis accordingly.


Indulge In Treats
