Shop With Us (Admin)

This project is an e-commerce site admin. The initial build of this project was done with the help of a tutorial done by Code with Antonio: https://www.youtube.com/watch?v=5miHyP6lExg&list=WL&index=3. Multiple stores can be created and customised. The stats on the home page and orders page are created when orders come through on the front store. This is currently being developed, changed and upgraded further for a start-up T-Shirt business, therefore the code will not be available to view.
